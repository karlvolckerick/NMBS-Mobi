Key Technology Choices
VoiceLive — wraps GPT Realtime with Azure deep noise suppression. +7 pp success rate in noisy environments vs standard Realtime API.
Semantic Kernel — manages agent tools, MCP server plugins, and the auto-injected handoff functions between agents.
Python + FastAPI — async WebSocket handling for low-latency audio routing. Entire app in a single Docker container.
Terraform IaC — full infrastructure reproducible. One command to stand up a new environment.
